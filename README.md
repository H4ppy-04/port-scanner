# Port Scanner

A blazingly fast port scanner written in Rust 🦀


## Features

 - Blazingly fast multithreading with _fast_ mode
 - Compile report at end of scan
 - Limits to 1024 ports
 - Custom address specification


## Usage

```
Usage: port-scanner [COMMAND]

Commands:
  scan  Scan port range
  help  Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
